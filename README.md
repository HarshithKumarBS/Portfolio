# Harshith Kumar B S — Portfolio

Premium personal portfolio website built with Vite + React.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Customization

- **Personal info**: Update links, email, and descriptions in each component under `src/components/`
- **Resume**: Place your `resume.pdf` in the `public/` folder
- **Projects**: Edit `src/components/Projects.jsx` with real GitHub/demo URLs
- **Certifications**: Edit `src/components/Certifications.jsx`
- **Contact email**: Update `harshith@example.com` in `Contact.jsx` and `Footer.jsx`

## Tech Stack

- Vite 5
- React 18
- Lucide React (icons)
- Inter font (Google Fonts)
- Pure CSS with CSS custom properties

## Design System

Colors, typography, and spacing are defined as CSS custom properties in `src/index.css`.
